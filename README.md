# 学生成绩管理系统

## 项目简介
本项目是一个基于 `C++` 的控制台学生成绩管理系统，适合作为数据结构或程序设计课程设计的基础项目。系统以单链表作为核心数据结构，实现了学生成绩信息的录入、查询、插入、修改、删除、统计、排序以及文件持久化等功能。

该项目包含基础版本与改进版本两个源码文件。其中改进版本在原有功能的基础上，进一步加入了输入校验、数据保存与读取、统计分析、排序显示等内容，使系统更加完整，能够满足课程作业中对“数据结构应用”和“程序综合设计”的基本要求。

## 功能说明
系统主要实现了以下功能：

- 登记学生成绩信息
- 查询指定学生成绩
- 插入新的学生记录
- 更新学生成绩
- 删除学生信息
- 显示全部学生信息
- 统计学生总数、平均分、最高分、最低分
- 按成绩从高到低排序显示
- 将数据保存到文件，并在程序启动时自动读取

## 数据结构设计
本项目采用**单链表**存储学生信息。每个结点保存一名学生的基本数据，并通过指针连接成链式结构。

### 学生结点包含的信息
- 学号
- 姓名
- 成绩
- 指向下一个结点的指针

### 采用链表的原因
- 便于动态插入和删除结点
- 能够体现数据结构课程中链式存储的应用
- 适合作为控制台管理系统的基础实现方式

## 主要实现思路
系统运行后通过菜单与用户交互，根据用户输入执行对应功能。

- 录入功能：创建链表并依次保存学生信息
- 查询功能：按学号遍历链表查找目标学生
- 插入功能：向链表中增加新的学生结点
- 修改功能：查找到目标学生后更新成绩
- 删除功能：调整前驱结点指针并释放目标结点内存
- 统计功能：遍历链表计算总人数、平均分、最高分和最低分
- 排序功能：对链表中的学生成绩进行降序排列显示
- 持久化功能：将数据写入文件，在程序启动时再从文件中恢复

## 文件说明
仓库中的主要文件如下：

- `学生管理系统.cpp`：基础版本，包含录入、查询、插入、删除、显示等基本功能
- `学生管理系统_改进版.cpp`：改进版本，增加成绩更新、统计、排序、输入校验、文件保存与读取等功能
- `学生管理系统.sln`：Visual Studio 解决方案文件
- `学生管理系统.vcxproj`：Visual Studio 项目文件

## 运行环境
- 操作系统：Windows
- 开发工具：Visual Studio
- 编程语言：C++

## 运行方法
### 方法一：使用 Visual Studio 运行
1. 打开仓库中的 `学生管理系统.sln`
2. 选择需要运行的源码版本
3. 编译并运行项目
4. 按照控制台菜单提示输入相应操作

### 方法二：直接编译 `.cpp` 文件
如果本地已安装支持 C++ 的编译器，也可以直接编译源码文件，例如选择改进版源码进行编译运行。

## 项目特点
- 使用链表组织学生数据，体现数据结构知识的实际应用
- 采用菜单驱动方式，便于交互式操作
- 增加输入合法性校验，提升程序稳定性
- 支持文件保存与读取，避免程序退出后数据丢失
- 功能较完整，适合作为课程设计或实验项目展示

## 不足与改进方向
当前项目主要面向课程作业和控制台演示，仍有进一步提升空间：

- 界面较简单，后续可扩展为图形界面版本
- 数据存储方式较基础，后续可接入数据库管理系统
- 代码可进一步模块化，提升可维护性与复用性
- 可增加按姓名查询、多条件统计、成绩区间分析等功能

## 课程设计总结
通过本项目，可以将链表、遍历、查找、插入、删除、排序、文件读写等知识点结合起来，完成一个较完整的学生成绩管理系统。该系统不仅锻炼了基础编程能力，也体现了数据结构在实际问题中的应用价值。
